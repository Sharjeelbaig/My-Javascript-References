# Tailwind setup react vite

```bash
npm i install tailwindcss @tailwindcss/vite
```

in vite.config.ts

```ts
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite' // 1) import this

export default defineConfig({
  plugins: [
    tailwindcss(), // 2) call this function
  ],
})
```
import in main css
```css
@import "tailwindcss";
```
